Quetzal Digital Developer Hub

Welcome to the Quetzal Digital developer experience stack. This lightweight documentation portal is served via the docs service inside the development Docker Compose file.

Explore the available surfaces while the stack is running:

  • API (NestJS)
  • Wallet Web (Vite)
  • Admin Web (Vite)
  • Prism Mock Server
  • Docs (this site)

Use make dev to launch the full developer stack or make mock when you only need the mock services.
